
Чтобы мы могли рассмотреть ваш запрос на включение доступа к API, пожалуйста, выполните следующие пункты:
Подтвердите номер телефона в профиле: https://immers.cloud/user/telephone/
Убедитесь, что в профиле указаны реальные ФИО.
На балансе должна быть сумма не менее, чем на 2 часа работы нужной конфигурации.
Сообщите, пожалуйста, нам задачи и программное обеспечение, которые планируете использовать, и цель, для которой вам нужен доступ к API.
После этого я создам для вас заявку.

-----------------------------------------------------

API immers.cloud - API для работы с данными личного кабинета immers.cloud, например, для получения информации о пользователе, балансе, расходах, списке и доступности конфигураций, квотах и т. д.
Описание API есть в документе: https://immers.cloud/documents/immerscloud-apiv2-0725.pdf
